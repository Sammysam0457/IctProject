External Css

Applications



  • Keeps styles separate from content, making maintenance easier.

  • use to ensure consistent styles across multiple pages of the webs

  • makes it easy to update and change the styles

Example code

   
            <!DOCTYPE html>
            <html>
            <head>
            <link rel="stylesheet" href="mystyle.css>
            </head>
            <body>
            
            <h1>This is a heading</h1>
            <p>This is a paragraph.</p>
            
            </body>
            </html>
